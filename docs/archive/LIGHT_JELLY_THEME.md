# 🌿 浅色果冻玻璃主题 - 青色薄荷风格

## ✨ 设计概览

全新的浅色主题，使用青色 (#81D8CF) 配合薄荷绿色系 (#E6F2EB, #F2FAF7)，打造清新、通透、充满活力的果冻玻璃质感。

## 🍬 果冻玻璃效果

### 1. 磨砂玻璃（导航栏）
```css
background: rgba(255, 255, 255, 0.7);
backdrop-filter: blur(20px) saturate(180%);
border: 1px solid rgba(129, 216, 207, 0.25);
box-shadow: 果冻阴影;
```

**特点：**
- 70% 白色透明度
- 20px 模糊 + 180% 饱和度
- 青色半透明边框
- 多层阴影营造深度

### 2. 卡片果冻效果
```css
background: linear-gradient(
  135deg,
  rgba(255, 255, 255, 0.95) 0%,
  rgba(230, 242, 235, 0.9) 50%,
  rgba(242, 250, 247, 0.85) 100%
);
border: 1.5px solid rgba(129, 216, 207, 0.35);
box-shadow: 多层果冻阴影 + 内光;
```

**特点：**
- 渐变背景（白色→薄荷绿→薄荷白）
- 青色半透明边框
- 顶部内光效果
- Hover 时弹性动画

### 3. 按钮果冻效果
```css
background: linear-gradient(
  135deg,
  rgba(129, 216, 207, 0.95) 0%,
  rgba(95, 201, 190, 0.9) 100%
);
border: 1px solid rgba(255, 255, 255, 0.5);
box-shadow: 青色阴影 + 白色内光;
```

**特点：**
- 青色渐变背景
- 白色半透明边框
- 光泽动画效果
- 点击时涟漪动画

## 🎭 果冻阴影系统

**特点：**

- 顶部白色内光（高光效果）
- 从小到大递进

## 🌊 动画效果

### 1. 弹性缓动（Elastic）
```css
transition: all 0.3s cubic-bezier(0.34, 1.56, 0.64, 1);
```
- 果冻般的弹性动画
- Hover 和点击时的回弹效果

### 2. 卡片 Hover
```css
transform: translateY(-4px) scale(1.01);
box-shadow: 阴影加深;
border-color: 青色加深;
```
- 向上浮起 4px
- 轻微放大 1%
- 阴影和边框增强

### 3. 按钮 Hover
```css
transform: translateY(-2px) scale(1.05);
/* 光泽扫过动画 */
```
- 向上浮起 2px
- 放大 5%
- 光泽从左上扫到右下

### 4. 进度条光泽
```css
/* 进度条内的光带持续滚动 */
animation: shimmer 2s infinite;
```
- 白色光带从左到右滚动
- 2 秒循环
- 营造流动感

### 5. 涟漪效果
```css
/* 按钮点击时从中心扩散 */
.btn:active::before {
  width: 300px;
  height: 300px;
}
```
- 从按钮中心扩散
- 白色半透明涟漪
- 0.4 秒动画

## 🌟 特色效果

### 1. 顶部内光（Inner Glow）
所有卡片和按钮顶部都有 1-2px 的白色内阴影，模拟光从上方照射的效果。

### 2. 双层外阴影（Double Shadow）
外阴影使用两层：
- 近距离阴影（模糊小、扩散小）
- 远距离阴影（模糊大、扩散大）

### 3. 渐变背景（Gradient BG）
所有背景都使用渐变而非纯色：
- 135° 对角线渐变
- 从浅到深的自然过渡

### 4. 毛玻璃模糊（Backdrop Filter）
支持的元素使用 `backdrop-filter: blur()`：
- 导航栏：20px 模糊
- 卡片：8-10px 模糊
- 输入框：无模糊（性能优化）

### 5. 旋转光晕（Rotating Halo）
统计卡片背景有旋转的青色光晕：
- 10 秒完整旋转一圈
- 径向渐变
- 中心亮，边缘透明

## ✅ 设计特点总结

### 视觉特征
✅ **清新通透** - 薄荷绿色系营造清爽感  
✅ **果冻质感** - 多层阴影和内光营造立体感
