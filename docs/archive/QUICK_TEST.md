# 快速测试指南

## 🌐 浏览器测试（推荐先测试）

```bash
cd ~/Documents/master-nihongo-ios/frontend
npm run dev
```

打开浏览器访问 http://localhost:5173

### 测试路径
1. 点击底部"我的"标签
2. 依次点击测试所有选项：
   - ✅ 账号和安全 → 修改密码 → 填写表单 → 保存
   - ✅ 个人信息 → 编辑信息 → 修改昵称 → 保存
   - ✅ 通知提醒 → 切换开关 → 设置时间
   - ✅ 设置 → 切换主题 → 调整学习目标
   - ✅ 隐私 → 开启云端同步（查看提示）
   - ✅ 关于和帮助 → 展开常见问题
   - ✅ 关于 Master Nihongo → 查看应用信息

## 📱 iOS 测试

### 首次运行需要切换 Xcode
```bash
sudo xcode-select --switch /Applications/Xcode.app/Contents/Developer
```

### 打开项目
```bash
cd ~/Documents/master-nihongo-ios/frontend
npm run ios
```

或手动打开：
```bash
open ~/Documents/master-nihongo-ios/frontend/ios/App/App.xcworkspace
```

### 在 Xcode 中
1. 选择目标设备（模拟器或真机）
2. 点击运行按钮 ▶️ (⌘R)
3. 等待应用启动
4. 按照浏览器测试路径测试所有功能
