# 个人信息功能 - 完全前端可编辑

## ✅ 最新更新

所有个人信息现在都可以在前端直接编辑和保存，无需后端！

---

## 📝 可编辑字段

### 1. **昵称** ✏️
- 点击"编辑信息"按钮
- 修改昵称（最多 20 字）
- 点击"保存"即可
- **存储位置**：iOS 本地存储

### 2. **个人简介** ✏️
- 点击"编辑信息"按钮
- 修改简介（最多 100 字）
- 点击"保存"即可
- **存储位置**：iOS 本地存储

### 3. **头像** 📸
- 点击头像或相机图标
- 从相册选择照片
- 自动保存（2MB 以内）
- **存储位置**：iOS 本地存储（Base64）

### 4. **学习目标** 🎯
- 点击目标等级旁边的编辑按钮 ✏️
- 输入自定义目标（最多 30 字）
- 例如：`N5 → N3` 或 `冲刺 N2` 或 `100 天突破 N1`
- 点击"保存"即可
- **存储位置**：iOS 本地存储

### 5. **学习时长** ⏱️（自动追踪）
- 每学习满 1 分钟自动记录
- 显示格式：`12 小时 34 分钟`
- **无需手动操作**

### 6. **学习天数** 📅（自动追踪）
- 每天首次学习自动 +1
- 显示：`已坚持 7 天`
- **无需手动操作**

## 💾 数据存储

### 存储方式
- **技术**：Capacitor Preferences API
- **底层**：iOS UserDefaults
- **格式**：JSON

### 存储键
```typescript
key: "user_profile"
```

### 数据结构
```typescript
{
  nickname: "Master 用户",
  bio: "正在学习日语中...",
  avatar: "data:image/jpeg;base64,...",  // Base64 图片
  targetLevel: "N5 → N3",                // 可自定义
  studyTimeMinutes: 156,                 // 自动追踪
  studyDays: 7,                          // 自动追踪
  lastStudyDate: "2026-06-15",           // 自动记录
  createdAt: "2026-06-15T10:00:00Z",
  achievements: ["新手", "学习者"]       // 自动解锁
}
```

## 🎯 设计理念

### 为什么全部前端可编辑？

1. **离线优先** 📴
   - 无需网络连接
   - 无需后端服务器
   - 数据保存在本地

2. **即时生效** ⚡
   - 修改立即保存
   - 无需等待同步
   - 无网络延迟

3. **隐私保护** 🔒
   - 数据不上传服务器
   - 完全本地存储
   - 用户完全控制

4. **简单可靠** 🎯
   - 不依赖后端
   - 不会因服务器故障失败
   - 用户体验流畅

```typescript
// 可选：云端同步
async function syncToCloud() {

    method: 'POST',
    body: JSON.stringify(profile)
  });
}

// 可选：从云端恢复
async function syncFromCloud() {
  const response = await fetch('/api/profile');
  const profile = await response.json();
  await saveUserProfile(profile);
}
```
