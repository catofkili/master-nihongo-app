# 个人信息功能更新

## ✅ 已完成的改进

### 1. 真实的数据持久化
之前个人信息只是前端状态，刷新后会丢失。现在：
- ✅ 所有信息保存到 Capacitor Preferences（本地存储）
- ✅ 应用重启后数据仍然保留
- ✅ 支持跨会话数据持久化

### 2. 头像上传功能 📸
- ✅ 点击头像或相机图标可选择照片
- ✅ 自动转换为 Base64 保存
- ✅ 限制文件大小为 2MB
- ✅ 支持所有图片格式（JPG, PNG, HEIC 等）

### 3. 真实的学习统计 📊
**自动追踪：**
- ✅ 学习时长：每学习满 1 分钟自动同步
- ✅ 学习天数：每天首次学习自动 +1
- ✅ 最后学习日期：自动记录

**显示位置：**
- 个人信息页面显示累计数据
- 格式化显示：`12 小时 34 分钟`

### 4. 成就系统 🏆
**4 个成就徽章：**
1. 🏆 **新手** - 默认已解锁
2. 📚 **学习者** - 默认已解锁
3. 🔥 **坚持者** - 连续学习 7 天解锁
4. ⭐ **大师** - 累计学习 100 小时解锁

**动态显示：**
- 已解锁：彩色高亮显示
- 未解锁：灰色半透明
- 底部显示：`已解锁 2 / 4 个成就`

### 5. 编辑体验优化 ✏️
- ✅ 保存按钮显示"保存中..."状态
- ✅ 字数统计实时显示（昵称 20 字、简介 100 字）
- ✅ 保存失败时弹出错误提示
- ✅ 取消编辑时恢复原始值

## 📱 数据存储结构

### UserProfile 接口
```typescript
{
  nickname: string;           // 昵称
  bio: string;                // 个人简介
  avatar?: string;            // Base64 头像
  targetLevel: string;        // 目标等级（如 "N5 → N3"）
  studyTimeMinutes: number;   // 学习总时长（分钟）
  studyDays: number;          // 学习天数
  lastStudyDate?: string;     // 最后学习日期 (YYYY-MM-DD)
  createdAt: string;          // 创建时间
  achievements: string[];     // 已解锁的成就
}
```

### Capacitor Preferences 存储键
- **键名**: `user_profile`
- **值**: JSON 字符串
- **位置**: iOS UserDefaults（iOS 原生存储）

## 📝 注意事项

1. **头像大小限制**：2MB，超过会提示错误
2. **学习时长精度**：精确到分钟（不足 1 分钟不记录）
3. **学习天数计算**：按日期（YYYY-MM-DD）去重，每天只计一次
4. **成就解锁**：自动检测，无需手动触发
5. **数据备份**：Preferences 数据在卸载应用时会丢失
