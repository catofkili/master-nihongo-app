# 📱 Master Nihongo - 日语学习应用

一款功能完善的离线日语学习 iOS 应用，支持单词学习、语法学习、复习队列、错题本等功能。

## ✨ 主要特性

### 🎯 核心功能
- **单词学习** - N5-N1 完整词库，支持假名、汉字、释义学习
- **语法学习** - 系统化语法点讲解，配合练习题
- **沉浸式学习** - 全屏专注学习模式
- **智能复习** - 基于记忆曲线的复习系统
- **错题本** - 自动收集错题，针对性复习
- **学习统计** - 详细的学习进度和成就系统

### 🎨 界面设计
- 深色/浅色主题切换
- 流畅的动画效果
- 响应式布局（支持桌面端）
- iOS 原生体验

### 💾 数据管理
- 完全离线使用
- 本地数据存储
- 支持数据导出/导入
- 学习进度云同步（可选）

## 📦 压缩包内容

### 1. `master-nihongo-web-only.tar.gz` (2.9M)
纯 Web 构建产物，可直接部署：
- `frontend/dist/` - 编译好的前端资源
- `nihongo.db` - 完整词汇/语法数据库 (3.0M)

**适用场景：**

### 2. `master-nihongo-app.tar.gz` (6.0M)
完整项目（含 iOS 配置）：
- `frontend/dist/` - Web 构建产物

- `backend/` - 后端 API（可选）

**适用场景：**

- 完整开发环境

## 🚀 快速开始

### 方式 1：浏览器预览（推荐）

```bash
# 解压
tar -xzf master-nihongo-web-only.tar.gz

# 启动本地服务器
cd frontend/dist
python3 -m http.server 8080

# 浏览器访问
open http://localhost:8080
```

### 方式 2：Xcode 运行（iOS 真机体验）

```bash
# 解压完整包
tar -xzf master-nihongo-app.tar.gz

# 打开 Xcode 项目
cd frontend/ios/App
open App.xcworkspace

# 在 Xcode 中选择模拟器或真机运行
```
